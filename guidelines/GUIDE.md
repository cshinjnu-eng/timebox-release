# TimeBox 使用指南

**TimeBox** 是一款并行任务计时 + 时间记录 + 数据分析工具，帮助你追踪每天时间的去向。

---

## 四个主页面

| 页面 | 功能 |
|------|------|
| **计时** | 实时运行多个任务计时器 |
| **待办** | 轻量待办事项管理 |
| **数据** | 历史记录查看与统计分析 |
| **时间轴** | 24 小时可视化时间分布 |

---

## 核心概念

### 定性标签（每个任务必选一个）

| 标签 | 说明 |
|------|------|
| 🔵 工作 | 职业相关任务 |
| 🟣 学习 | 主动学习、读书、课程 |
| 🟢 生活 | 饮食、运动、休闲 |
| 🟠 琐事 | 杂务、日常事务 |
| 🩵 睡觉 | 睡眠时间 |

### 评估标签（可选，用于事后反思）

| 标签 | 含义 |
|------|------|
| 必须 / 有意义 | 该做，且有价值 |
| 必须 / 没意义 | 该做，但感觉无意义 |
| 不必须 / 有意义 | 可做可不做，但有收获 |
| 不必须 / 没意义 | 可做可不做，且没价值 |

> 评估标签可在任务创建时预设，也可在结束时回顾。

---

## 开始计时

1. 点右上角 **「新建」** 按钮
2. 填写任务名称（必填）
3. 选择 **定性标签**（必填）和 **评估标签**（可选）
4. 可选填：预计时长（分钟）、自定义标签（如 `#项目A`）、描述
5. 点「开始」→ 任务立即计时

> 支持同时运行多个任务，互不干扰。

---

## 停止任务

1. 点任务卡片上的 **■ 停止**按钮
2. 弹窗显示本次用时，可填写「完成感受」
3. 点「确认」→ 保存为一条工作记录

---

## 暂停与恢复

- 点 **⏸ 暂停**：计时器暂停，任务保留
- 点 **▶ 继续**：恢复计时
- 暂停中的任务不计入当前用时

---

## 手机悬浮窗（Android）

离开 App 后，屏幕上会浮现一个小窗口：

- **点 header 区域**（标题行）→ 收起 / 展开任务列表
- **点任务行**→ 返回 App
- **点 ✕**→ 关闭悬浮窗并停止所有计时

> 需在系统设置中授予「显示在其他应用上方」权限才能使用悬浮窗。

---

## 手动补录记录

适用于事后补充未记录的时间段：

1. 进入 **数据** 页
2. 点「**手动记录**」
3. 填写任务名、分类、开始时间、结束时间
4. 提交即可

---

## 数据页

- **学习/工作** 卡片：仅统计「工作」和「学习」分类的总时长
- **时间分布图**：按小时展示当天时间分布，可切换「定性标签」/ 「评估标签」两种配色
- **历史记录**：按日期分组，支持搜索和分类过滤
- **日报**：一键生成当天工作总结
- **导出 CSV**：右上角按钮，导出全部记录

---

## 时间轴

- 纵轴为 0:00–24:00 的完整一天
- 每条工作记录显示为色块，颜色对应定性标签
- 时间重叠的任务自动**并排**显示
- 左上角可按日期前后切换
- 顶部可按分类过滤
- 底部显示分类汇总、记录列表
- Android 设备额外显示**屏幕使用时间**（需授权）

---

## 待办事项

- 点「**+**」快速添加待办
- 可设置优先级（高 / 中 / 低）和分类
- 点圆圈标记完成，顶部显示完成进度
- 独立于计时任务，适合轻量清单管理

---

## 数据安全

- 所有数据保存在**本地设备**（IndexedDB）
- 完全离线，不联网，不上传任何数据
- 可随时通过「导出 CSV」备份数据
