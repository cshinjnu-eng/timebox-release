# TimeBox

Android 时间管理应用，帮助你记录、追踪和反思每天的时间使用。

## 功能

### 计时系统
- 并行任务计时器，支持多任务同时运行
- 全局悬浮窗，离开 App 后持续显示计时
- 任务分类（工作/学习/生活/娱乐）+ 四象限评估标签
- 自定义标签系统，智能自动补全
- 手动补录、自定义开始时间

### 点子管理
- 快速捕获灵感，随时记录
- 点子评估打分（影响力/可行性/兴趣/紧迫性）
- 生命周期管理：收件箱 → 评估 → 立项 → 完成/归档
- 里程碑 + 子任务拆分，支持从子任务直接启动计时
- 计时结束自动回写时间到点子，追踪时间投入

### 数据分析
- 24h 时间轴（任务甘特图 + 屏幕使用双列）
- 每日日报（工时/屏幕使用/睡眠/待办统计）
- 工作记录历史，CSV 导出

### 其他
- 待办清单
- 应用桶检测：自动感知刷手机时长并提醒计时
- 睡眠自动检测
- 长线任务挂载

## 安装

1. 前往 [Releases](../../releases) 下载最新 APK
2. 手机开启「允许安装未知来源应用」
3. 安装后授权：**悬浮窗权限** + **使用情况访问权限**

## 系统要求

Android 8.0 (API 26) 及以上
