# TimeBox

Android 时间管理应用，帮助你记录、追踪和反思每天的时间使用。

## 功能

- 并行任务计时器，支持多任务同时运行
- 全局悬浮窗，离开 App 后持续显示计时
- 任务分类 + 四象限评估标签
- 时间轴（任务甘特图 + 屏幕使用双列）
- 每日日报（工时 / 屏幕使用 / 睡眠 / 待办）
- 待办清单、工作记录、手动补录、CSV 导出
- 应用桶检测：自动感知刷手机时长并提醒计时

## 安装

1. 前往 [Releases](../../releases) 下载最新 APK
2. 手机开启"允许安装未知来源应用"
3. 安装后授权：**悬浮窗权限** + **使用情况访问权限**

## 系统要求

Android 8.0 (API 26) 及以上
