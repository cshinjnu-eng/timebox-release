## 功能概览

### 计时与记录
- **并行多任务计时** — 同时运行多个计时器，工作、学习、生活分类管理
- **悬浮窗** — 后台悬浮显示计时，切换 App 不中断
- **手动补录** — 支持补录过去任意时段的时间记录
- **长线任务** — 跨天持续追踪的长期项目，支持里程碑和检查点

### 应用桶（自动检测）
- 自动识别手机使用记录（微信、抖音、B站等），将碎片化使用时间聚合补录
- 可自定义应用分组、触发阈值、容忍间隔

### 点子管理
- 点子从 Inbox → 评估 → 执行 → 完成的完整生命周期
- 可评估可行性、必要性、影响力，自动计算综合得分
- 点子子任务直接启动计时，时间投入与点子绑定

### 待办
- 优先级（高/中/低）+ 分类管理
- 归档而非删除，保留历史记录

### 数据分析
- 24h 时间轴可视化，支持按分类过滤、按天翻页
- 历史记录与 CSV 导出
- 屏幕使用时间展示（Android 使用权限）

### AI 智能助手
- **自然语言操作** — 说「昨晚9点到11点看了2小时书」自动补录记录
- **日报 / 周报生成** — 结构化报告，含效率评分和改进建议
- **时间分析** — 发现碎片化、娱乐过多等问题并给出行动建议
- **成长洞察** — 结合人格档案给出个性化深度分析
- 支持多轮对话，上下文连贯
- 需用户自备阿里百炼 API Key（免费额度足够日常使用）

### 个人档案 · 属性成长系统
- **六边形战士雷达图** — 执行力 / 创造力 / 专注力 / 自控力 / 求知欲 / 抗压性，基于行为数据动态计算（EMA 时间衰减）
- **MBTI** — 支持全部 16 种类型，含人格分组和特质描述
- **大五人格（OCEAN）** — 滑块导入测试结果，AI 分析时作为人格背景参考
- **特质标签** — 根据行为自动解锁（晨曦守望者、深度专注者、七日连击……）

## 技术栈

| 层级 | 技术 |
|------|------|
| 前端框架 | React 18 + TypeScript + Vite 6 |
| 样式 | Tailwind CSS v4 + 内联 style（深色主题） |
| 移动端 | Capacitor 8 → Android 原生 Java |
| 数据存储 | IndexedDB（本地，无需账号） |
| AI | 阿里百炼 DashScope（用户自带 API Key，数据不上传） |

## 隐私说明

所有数据存储在**设备本地**（IndexedDB），不联网、不注册账号。AI 功能需用户自行填入 API Key，请求直接从设备发往 AI 平台，不经过任何中间服务器。
